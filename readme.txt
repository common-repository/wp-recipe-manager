== Description ==
WP Recipe Manager plugin supports fully features for a cooking recipe with recipe general information, ingredients/ instruction creation. Just by drag & drop, you can create and order the ingredients, instruction steps quickly. 

An outstanding feature is the ability to mark "completed" Ingredients & Instructions in frontend. Very flexible.


== Features ==
* Create Recipes, Categories, Tags and Cuisines.
* Drag & drop ingredients and instruction steps.
* Click to mark complete Ingredients & Instructions (frontend).
* Allow overwrite template from theme or child-theme.
* Allow set featured image to recipe.

== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/wp-recipe-manager/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Go Settings -> Permalinks, Click to "Save Changes" button to rebuild permalinks.
